# 機能一覧

| 大分類 | 機能名 | 詳細・仕様 | 優先度/備考 |
| :--- | :--- | :--- | :--- |
| **基盤** | **Docker実行環境** | 個人用PC上でWebサーバーとDBサーバーを稼働させる。 | 必須 |
| | **LAN内アクセス** | 同一Wi-Fi内のスマホや他PCからアクセス可能にする。 | 必須 |
| **データ管理** | **カード登録** | 以下の項目を保存する。<br>・名称<br>・属性 (水, 雷など)<br>・進化度合い (たね, 1進化など)<br>・特徴 (exなど)<br>・わざの属性 (複数選択可)<br>・枚数<br>・メモ | 必須 |
| | **カード紐付け** | 登録済みカードどうしで、進化元・進化先の紐付けを行う。 | 必須 |
| | **カード編集・削除** | 登録済みカードの情報を修正、または削除する。 | 必須 |
| | **在庫数増減** | 検索後のリストから直接、枚数を「+1」「-1」する。 | 必須 |
| | **管理項目追加** | 登録できるカード情報項目を追加できる。 | 中 |
| | **画像登録** | カードの写真をアップロードし、紐付ける。 | 低 |
| **検索・閲覧** | **一覧表示** | 登録カードをリスト形式で表示する。スマホ対応 (レスポンシブ)。 | 必須(レスポンシブは中) |
| | **キーワード検索** | カード名称などで検索する。 | 必須 |
| | **フィルタ検索** | 属性、進化度合い、特徴などで絞り込む。 | 必須 |
| **入出力** | **CSVエクスポート** | DB内のデータをCSV形式でダウンロードする。 | 必須 |
| | **CSVインポート** | (提案機能) CSVファイルを取り込み、一括登録・更新を行う。 | 中 |
