プライバシーポリシー (Privacy Policy)

本ドキュメントは、P TCG Home Gym Manager（以下「本サービス」といいます）の個人情報の取り扱いについて定めるものです。
ユーザーの皆様（未成年者の場合はその親権者等の法定代理人を含みます）におかれましては、本サービスを利用する前に必ず本ドキュメントをご確認いただき、同意の上でご利用ください。
本サービスを利用することにより、本規約のすべての条件に同意したものとみなされます。

第1条（収集する情報）

運営者は、本サービスの提供にあたり、以下の情報をユーザーから取得します。

  1. アカウント情報: 登録用メールアドレス、パスワード（暗号化して保存）、ユーザーID。
  2. ユーザーコンテンツ: ユーザー自身が撮影・アップロードしたカード画像、およびユーザーが登録したコレクションデータ（カード名、所持数、メモ等）。
  3. 利用状況: アクセスログ、Cookie、利用端末情報、IPアドレス。

第2条（利用目的）

取得した情報は、以下の目的で利用されます。

  1. 本サービスの提供、本人確認、設定の保存のため。
  2. AI画像解析: アップロードされた画像をAIに解析させ、テキスト情報を抽出するため。
  3. サービス改善: 利用統計の作成、バグの発見・修正のため。
  4. 広告配信: ユーザーの興味最適化された広告を表示するため（Cookieを利用）。

第3条（画像データとAI学習について）

  1. 通常利用: ユーザーがアップロードした画像は、本サービスの機能（表示・解析・バックアップ）を提供するために必要な範囲で、運営者が契約するクラウドサーバー（Google Cloud等）に保存・処理されます。
  2. 第三者APIへの提供: AI解析を行うため、ユーザーの画像データは一時的に Google LLC (Gemini API / Vertex AI) へ送信されます。
    • 本サービスではGoogle Cloudの有料版API（Enterpriseプラン等）を利用しており、原則として、ユーザーの画像データがGoogleのAIモデルの学習（入力データの再利用）に使用されることはありません。（Google Cloud利用規約に準拠）
  3. AI学習への同意（オプトイン）:
    • 本サービスの設定において「AI精度の向上のために画像データを提供する」旨の同意を行ったユーザーに限り、運営者は、その画像データを匿名化した上で、本サービスのAIモデルの再学習（ファインチューニング）や精度検証のために利用できるものとします。

第4条（情報の第三者提供）

運営者は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  1. ユーザーの同意がある場合。
  2. 法令に基づく場合。
  3. 広告配信事業者への提供: 広告配信のために、Cookie等の識別子をGoogle AdSense等の広告事業者が取得・利用する場合があります。

第5条（データの保存期間と退会）

  1. 保存期間: ユーザーがアカウントを保持している限り、登録データは原則として永続的に保存されます。
  2. 退会時の処理: ユーザーが退会手続きを行った場合、運営者は速やかにサーバー上の個人データ（メールアドレス、画像データ等）を削除、または個人を特定できない状態に加工します。ただし、システムのバックアップに残存するデータについては、一定期間経過後に自動的に消去されます。

第6条（プライバシーポリシーの変更）

本ポリシーの内容は、法令改正や本サービスの変更に伴い、予告なく変更されることがあります。変更後のポリシーは、本サービス上に掲載された時点で効力を生じるものとします。
